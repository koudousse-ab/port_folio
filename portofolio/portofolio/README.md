# Portfolio Project

## Description
This project is a personal portfolio showcasing the skills and projects of TCHEDRE Aboudoul-Koudousse. It includes sections for about, skills, and contact information, along with a profile picture and a modern design that supports both light and dark modes.

## Project Structure
```
portofolio
├── assets
│   ├── images
│   │   └── profile.jpg
├── css
│   └── styles.css
├── index.html
└── README.md
```

## Files Overview

- **assets/images/profile.jpg**: This is the profile picture that will be displayed on the portfolio.
  
- **css/styles.css**: This file contains the styles for the portfolio, including styles for light and dark modes, as well as variations in colors for different sections of the portfolio.

- **index.html**: The main HTML document for the portfolio. It includes sections for the user's information, skills, and contact details, and will be updated to include the profile picture and toggling between light and dark modes.

## Setup Instructions

1. **Clone the Repository**
   ```bash
   git clone <repository-url>
   cd portofolio
   ```

2. **Open the Project**
   Open the `index.html` file in your web browser to view the portfolio.

3. **Toggle Light/Dark Mode**
   The portfolio supports light and dark modes. You can toggle between them using the provided button in the header.

4. **Update Profile Picture**
   To change the profile picture, replace the `profile.jpg` file in the `assets/images` directory with your desired image, ensuring it retains the same filename.

5. **Customize Styles**
   Modify the `css/styles.css` file to change colors, fonts, and other styles as needed to personalize your portfolio.
